● SOCIAL MEDIA WEBSITE ( CODEIAL )
➢ A social website uses social and manual authentication using passportjs strategies.
➢ Contains a timeline that shows the posts. Allows to post a para on which other users can like and
comment.
➢ Getting notifications via mails using nodemailer module. Getting flash messages using connect-flash
and express-session.
➢ Allows to add and update details on your profile and can upload a profile picture using multer
middleware. Allows you to add other users as friend and can see their profiles.
➢ A group chat engine for all the users using socket.io.
➢ Performed Broken Access Control attack and resolved the common vulnerabilities on Social Lite.
